## 1. 安装

听打 Android 版以 `.apk` 文件直接分发（暂未上架应用商店）。

1. 在手机上打开 [最新 Android 版本](https://github.com/hungryalgo/starlingtype-releases/releases/tag/android-v0.1.0)，下载 **`StarlingType-Android-0.1.0.apk`**。
2. 点按下载的文件。Android 会询问是否允许您的浏览器（或"文件"应用）**安装未知应用**——允许一次即可继续。
3. 点 **安装**，再点 **打开**。

> 这一"安装未知应用"步骤对于应用商店之外分发的应用是正常的，只需操作一次。

**要求：** Android 8.0 或更新。首次运行会下载约 **280 MB** 的本地语音模型（建议在 Wi-Fi 下）。支持手机与折叠屏。

## 2. 首次设置

首次打开听打时，会有一个简短的向导引导您完成全部设置：

1. **欢迎** — 简要介绍。
2. **下载语音模型（约 280 MB）** — 点 **下载**。仅需一次，建议在 Wi-Fi 下。等待出现"✓ 模型已安装"。
3. **麦克风** — 点 **允许麦克风**。听打仅在您听写时聆听，音频绝不离开手机。
4. **启用键盘** — 点 **启用键盘**，在系统列表中打开 **StarlingType**，然后返回。*（仅"输入到任意应用"方式需要——见 [第 6 节](#6-切换到听打键盘)。）*
5. **全部就绪** — 点 **完成**，进入主屏幕。

随时可在 **设置 → 重新运行设置** 再次进行。

## 3. 三种听写方式

听打提供三种听写方式，按当下需要选择。

### A. 点按听写（无需设置）— 复制到剪贴板
适合快速记下一个想法，再粘贴到别处。

1. 打开 **听打** 应用。
2. 点 **🎤 点按听写**。
3. 听到提示音后说话。
4. 文字会**复制到剪贴板**——切到任意应用粘贴。

### B. 听打键盘 — 直接输入到文本框
适合在其他应用中直接书写（消息、邮件、笔记）。

1. 在任意应用中点按文本框，调出键盘。
2. **切换到听打键盘**（见 [第 6 节](#6-切换到听打键盘)）。
3. 点 **🎤 听写** 并说话 → 文字会**输入到该文本框**。
4. 还有一个 **⌨ 切换键盘** 按钮可切回常用键盘。

> 听打键盘是**纯语音**键盘（没有字母键）。请将常用键盘保留为默认，仅在听写时切换到听打。

### C. 快捷设置磁贴 — 随时随地、免手动
适合免手动听写。这是**不打开键盘也能使用唤醒词的唯一方式。**

1. 将 **StarlingType** 磁贴添加到快捷设置（从屏幕顶部下滑出现的面板）。最简单的方式：在应用中 **使用方法 → 添加快捷设置磁贴** → 确认。
2. 点磁贴将其 **开启**——听打开始在后台聆听。
3. 说一个 **唤醒词**（见下文），再说话。
4. 文字会**复制到剪贴板**以供粘贴。
5. 再点磁贴即可关闭。

## 4. 唤醒词

唤醒词告诉听打"开始聆听"。先说一个，再说话。

| English | 中文 |
|---|---|
| "Hey Type" | 嘿小听 |
| "Hey Starling" | 小听小听 |
| | 嘿听打 |
| | 听打听打 |

**唤醒词在哪些情况下可用：**
- ✅ **快捷设置磁贴**开启时（免手动，任意应用中）。
- ✅ 在**听打键盘**中，*如果*您启用了 **设置 → "键盘打开时聆听唤醒词"**（更耗电）。
- 其他情况下，直接**点按** 🎤 即可，无需唤醒词。

> 提示：说完唤醒词稍作停顿（您会听到提示音），以免句子开头被截断。

## 5. 设置

从主屏幕打开 **设置**。

- **语言** — 跟随系统 / English / 中文（见 [第 7 节](#7-语言english--中文)）。
- **键盘复制到剪贴板（而非直接输入）** — *默认关闭。* 关闭时键盘直接输入到文本框。仅当某个应用不接受直接输入时才开启；此时键盘上会出现 **📋 粘贴** 按钮以插入文字。
- **提示音** — 录音开始/结束时的声音。
- **振动反馈** — 触觉反馈。
- **键盘打开时聆听唤醒词** — *默认关闭。* 让您在键盘打开时无需点 🎤，直接说唤醒词。⚠️ 更耗电（键盘打开期间麦克风一直开启）。
- **唤醒词灵敏度** — 越高越不易误触发；越低越容易触发。下次开始聆听时生效。
- **电池优化** — 若想让常驻听写更稳定，可打开系统设置页。

## 6. 切换到听打键盘

Android 只允许**您本人**（而非应用）切换当前键盘。两种简便方式：

**最简单：** 在听打应用主屏幕点 **⌨ 切换到 StarlingType 键盘**——系统选择器会弹出，选 **StarlingType**。

**输入时：** 当键盘显示时，点导航栏（通常在右下角）出现的**键盘图标**，选择 StarlingType。

> **三星手机：** 导航栏上的键盘图标通常默认关闭。请在 **设置 → 通用管理 → 键盘列表和默认键盘 → 导航栏上的键盘按钮** 中开启。开启后，在任意应用中一点即可切换。

听写完成后，用键盘上的 **⌨ 切换键盘** 按钮（或同一个导航栏图标）切回常用键盘。

## 7. 语言（English / 中文）

整个应用——按钮、引导、键盘、通知——均支持 **English** 与 **简体中文**。

在 **设置 → 语言** 中更改：
- **跟随系统** — 跟随手机语言。
- **English**
- **中文**

更改会立即应用到整个应用，无需重启。

## 8. 疑难解答

**键盘麦克风没有输入任何文字。** 请确认 **设置 → "键盘复制到剪贴板"** 处于**关闭**状态（默认），这样才会直接输入。若开启了，听写会进入剪贴板——点 **📋 粘贴** 插入。

**唤醒词没反应。** 唤醒词仅在**快捷设置磁贴开启**时，或键盘打开且启用了"键盘打开时聆听唤醒词"时才生效。其他情况下请点 🎤。

**"切换键盘"按钮似乎没反应。** 点一下并稍等系统选择器弹出。若手机隐藏了导航栏键盘图标，请改用应用内的 **⌨ 切换到 StarlingType 键盘** 按钮。

**模型无法下载。** 一次性约 280 MB 的下载需要网络连接。请尝试 Wi-Fi，再到 **设置 → 重新运行设置** 重试。

**常驻（磁贴）没有通知。** 如果您拒绝了通知权限，后台服务仍会运行并仍会把听写复制到剪贴板，只是没有可见通知。您可稍后在系统应用设置中授予通知权限。
