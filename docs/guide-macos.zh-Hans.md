## 1. 安装（约 1 分钟）

1. 从 [macOS 版本](https://github.com/hungryalgo/starlingtype-releases/releases/tag/v0.1.1) **下载** `StarlingType-0.1.1.dmg`。
2. **打开** `.dmg`，将 **StarlingType** 拖入 **应用程序**。
3. **仅首次：** 右键（或按住 Control 点按）应用 → **打开** → 确认。这是一次性步骤——见 [第 7 节](#7-gatekeeper-警告)。

---

## 2. 首次设置

首次启动时会运行设置向导，引导您完成：

1. **欢迎与隐私** — 简要介绍；音频留在 Mac 上。
2. **麦克风** — 允许后听打才能听到您说话。
3. **模式** — *光标模式*（推荐）将文字输入到任意应用的光标处；*剪贴板模式*将其复制供您粘贴。
4. **辅助功能 / 自动化**（仅光标模式）— 授予后听打才能把文字插入光标处。向导会在每个系统提示出现前先做说明。
5. **唤醒词实测** — 试一个唤醒词，看到"我们听到了"。
6. **完成** — 可选启用 **登录时启动**（推荐，因为它是菜单栏应用，没有程序坞图标）。

---

## 3. 如何使用

运行后，听打位于您的 **菜单栏**（一个麦克风图标）。

1. 说一个 **唤醒词**（见下文）。
2. 听到 **pop** 提示音——这是开口的信号。
3. 说出您的句子，中英文随意混说。
4. 停止后会响起 **tink** 提示音，文字出现在**光标处**（或 **剪贴板**，取决于您的模式）。

不想说话？点菜单栏面板里的 **立即听写**，无需唤醒词即可听写。

---

## 4. 唤醒词

**English：**
- "Hey Type"
- "Hey Starling"

**中文：**
- 嘿小听
- 小听小听
- 嘿听打
- 听打听打

> 提示：说完唤醒词稍作停顿（提示音是信号）。若不停顿地说"Hey Type hello world"，可能被识别成"Type hello world"。

## 5. 设置

从菜单栏图标打开齿轮面板：

- **模式** — 光标（输入到光标处）或剪贴板（复制以粘贴）。
- **麦克风设备** — 按名称选择输入设备（AirPods 断连重连也不受影响）。
- **唤醒灵敏度** 与 **VAD** 调节。
- **提示音** — 开关 pop/tink 声音。
- **语言** — 跟随系统 / English / 中文，用于应用自身界面。
- **登录时启动** — 无需每次手动打开即可随时可用。
- **恢复快捷方式** — 若需要重新授权，可快速打开辅助功能/自动化面板。

---

## 6. 系统要求

- **macOS Ventura (13.0) 或更新。**
- **Apple Silicon（M1 或更新）。** 不支持 Intel Mac——本地语音引擎仅提供 Apple Silicon 二进制。
- 约 450 MB 磁盘空间（主要是语音识别模型）。
- 可用的麦克风。

## 7. Gatekeeper 警告

听打未使用 Apple Developer ID 签名（该计划每年 99 美元；为了免费的 v1.0，我们选择不将此成本转嫁给用户）。因此 macOS 首次启动会显示"无法打开 StarlingType，因为它来自身份不明的开发者"。

**右键应用 → 打开** 可一次性绕过；之后 macOS 会记住您的选择。若使用量足够，未来版本会签名并公证，此警告将消失。

---

## 8. 更新后的权限

macOS 将权限授予（麦克风、辅助功能）与应用的代码签名绑定。听打每个版本都使用**相同的身份**签名，因此您的授权会**在更新后保留**——通常无需重新授权。

万一某项权限丢失，听打会在启动时检测到并提供一键 **重新运行设置**。手动恢复（极少需要，约 20 秒——顺序很重要）：

1. 系统设置 → 隐私与安全性 → **辅助功能**（或齿轮面板的 **打开辅助功能**）。
2. **选中 StarlingType**，点 **−** 按钮移除该条目。
3. **退出听打。**
4. 从应用程序重新启动。
5. 系统提示时授予权限。

请勿跳过退出 + 重启——在旧进程仍运行时重新添加条目会留下缓存的"已拒绝"状态。

## 9. 已知问题

- **首次启动 Gatekeeper 警告** — 右键 → 打开一次（见 [第 7 节](#7-gatekeeper-警告)）。
- **唤醒词可能混入您的第一句话**，若不停顿——请停顿约 300 毫秒（提示音是信号）。
